# Excel to json

## excel format

| key | language-1  | language-2   |   |   |
|-----|-------------|--------------|---|---|
| id  | language-id | language2-id |   |   |
|     |             |              |   |   |
|     |             |              |   |   |

| key | en  | tr   | 
|-----|-------------|--------------|
